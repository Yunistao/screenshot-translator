# 截图翻译工具 - Snipaste 风格重构设计

> 日期：2026-03-19

## 一、需求概述

将现有截图翻译工具重构为类似 Snipaste 的体验，主要包括：

1. **自由区域选择** — 热键触发后，用户框选截图区域
2. **底部工具栏** — 语言设置、翻译、编辑、置顶、取消等功能
3. **智能翻译** — 只翻译指定源语言的文字，其他语言保持原样
4. **标注功能** — 矩形框、箭头、画笔、文字
5. **置顶窗口** — 可拖动、可缩放的 Always-on-top 预览窗口
6. **LLM 翻译支持** — 新增 OpenAI、Claude、Gemini 三大 LLM 翻译引擎

## 三、截图流程

**用户操作流程：**

```
按下热键 (如 F1 或 Alt+S)
        │
        ▼
┌───────────────────────┐
│   创建全屏覆盖窗口     │  ← 透明背景，捕获鼠标事件
│   鼠标变为十字光标     │
└───────────────────────┘
        │
        ▼
用户拖动鼠标框选区域
        │
        ▼
┌───────────────────────┐
│   显示选区边框        │  ← 虚线/实线边框
│   显示选区尺寸        │  ← 如 "320 x 240"
│   显示底部工具栏      │
└───────────────────────┘
        │
        ▼
用户可选择：
  ├─ 点击"翻译" → OCR + 翻译 → 显示结果
  ├─ 点击"编辑" → 进入标注模式
  ├─ 点击"置顶" → 创建置顶窗口
  ├─ 点击"取消" 或 按 ESC → 关闭截图窗口
  └─ 按 Enter → 确认截图（可配置默认行为）
```

**底部工具栏按钮（从左到右）：**
- 语言选择下拉菜单（EN→CN / JA→CN 等）
- 翻译按钮
- 编辑按钮（矩形/箭头/画笔/文字）
- 置顶按钮
- 取消按钮 (X)

## 四、编辑功能

**编辑工具栏（点击"编辑"后展开）：**

```
┌────────────────────────────────────────────────────┐
│  [矩形] [箭头] [画笔] [文字] [撤销] [重做] [完成]  │
└────────────────────────────────────────────────────┘
```

**各工具行为：**

| 工具 | 操作方式 | 样式 |
|------|----------|------|
| 矩形 | 拖动绘制矩形框 | 红色边框，2px |
| 箭头 | 拖动绘制箭头线 | 红色，末端箭头 |
| 画笔 | 自由绘制线条 | 红色，2px，平滑曲线 |
| 文字 | 点击位置后输入文字 | 红色字体，可拖动位置 |

**编辑交互：**
- 选择工具后，鼠标变为对应图标
- 绘制完成后自动选中该标注，可拖动调整位置
- 选中标注时显示删除按钮，可删除
- 撤销/重做支持最近 20 步操作
- 点击"完成"退出编辑模式，回到主工具栏

**颜色方案：**
- 默认红色（#FF0000）
- 可在设置中自定义标注颜色

## 五、翻译引擎

**翻译引擎架构：**

```
┌─────────────────────────────────────────────────────┐
│                   翻译服务接口                       │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐  │
│  │Microsoft│ │ Google  │ │ Baidu   │ │ Youdao  │  │
│  └─────────┘ └─────────┘ └─────────┘ └─────────┘  │
│  ┌─────────┐ ┌─────────┐ ┌─────────┐             │
│  │ OpenAI  │ │ Claude  │ │ Gemini  │  ← 新增     │
│  └─────────┘ └─────────┘ └─────────┘             │
└─────────────────────────────────────────────────────┘
```

**LLM 配置项：**

| 服务商 | 必填配置 | 可选配置 |
|--------|----------|----------|
| OpenAI | API Key | Base URL（默认官方）、模型（gpt-4o/gpt-3.5-turbo） |
| Claude | API Key | Base URL、模型（claude-3-sonnet/claude-3-haiku） |
| Gemini | API Key | 模型（gemini-pro） |

**LLM 翻译 Prompt 模板：**

```
你是一个翻译助手。请将以下图片中的 {源语言} 文字翻译成 {目标语言}。
要求：
1. 只翻译 {源语言} 文字，其他语言的文字保持原样
2. 保持原文的格式和换行
3. 只返回翻译结果，不要解释

图片中的文字：
{OCR识别的文字}
```

**智能语言过滤逻辑：**
1. OCR 识别出所有文字及对应语言
2. 筛选出匹配用户设置源语言的文字片段
3. 只将筛选后的文字发送给翻译引擎
4. 合并翻译结果：翻译内容 + 原样保留的其他语言

## 六、置顶窗口

**置顶窗口特性：**

```
┌─────────────────────────────────┐
│  [最小化]            [关闭] ×   │  ← 标题栏（可拖动）
├─────────────────────────────────┤
│                                 │
│      截图预览（可缩放）          │
│                                 │
├─────────────────────────────────┤
│  原文: Hello World              │  ← 可折叠面板
│  译文: 你好世界                 │
└─────────────────────────────────┘
```

**窗口行为：**

| 功能 | 说明 |
|------|------|
| 拖动 | 拖动标题栏移动窗口位置 |
| 缩放 | 拖动窗口边角调整大小，图片自适应 |
| 最小化 | 收缩为小图标，贴边隐藏 |
| 关闭 | 销毁窗口，释放资源 |
| 置顶 | 始终保持在所有窗口最前方 |

**默认尺寸与限制：**
- 最小尺寸：200 x 150 px
- 最大尺寸：屏幕 80%
- 默认尺寸：截图原始大小（上限 400 x 300）

**快捷操作：**
- 双击图片 → 放大/缩小切换
- 右键图片 → 复制图片 / 复制译文 / 重新翻译

## 九、关键技术点

| 功能 | 技术方案 |
|------|----------|
| 全屏覆盖窗口 | Electron BrowserWindow，transparent: true, frame: false |
| 区域选择 | Canvas 绘制选区边框 + 鼠标事件监听 |
| 标注绘制 | Canvas 2D API，支持撤销/重做栈 |
| 置顶窗口 | BrowserWindow alwaysOnTop: true |
| 语言检测 | Tesseract.js 内置语言置信度 |
